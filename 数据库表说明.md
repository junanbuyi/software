# 数据库表说明

## 一、电价预测平台（预测页面）

以下表用于电价预测平台的各功能模块：

| 表名 | 说明 | 用途 |
|------|------|------|
| `admin_users` | 管理员用户 | 登录认证 |
| `base_price_data` | 基础电价数据 | 校核对比、图表展示 |
| `base_prediction_data` | 基础预测数据 | 预测基准数据 |
| `datasets` | 数据集信息 | 数据管理页面 |
| `dataset_files` | 数据集文件 | 数据集上传文件记录 |
| `dataset_records` | 数据集记录 | 数据集具体数据行 |
| `models` | 模型信息 | 模型管理页面 |
| `plans` | 预测方案 | 预测管理页面 |
| `plan_results` | 方案结果 | 预测结果存储 |
| `prediction_details` | 预测详情 | 预测详情页面 |
| `price_predictions` | 电价预测结果 | Mamba确定性预测数据 |
| `rankings` | 评分排名 | 评分管理页面 |
| `tcn_prob_predictions` | TCN概率预测 | TCN概率预测数据 |

## 二、电力市场平台（市场页面）

以下表用于电力市场平台，数据来源于 `电力市场平台数据/` 目录下的三个Excel文件：

### 来源: ROTS 44_DM&ASInputData.xlsm（输入配置数据）

| 表名 | 说明 | 来源Sheet | 记录数 | 对应页面模块 |
|------|------|-----------|--------|-------------|
| `market_thermal_plants` | 火电厂信息 | UnitThermalPlants | 8 | 企业信息 |
| `market_thermal_units` | 火电机组参数(23项) | UnitThermalGenerators | 32 | 企业信息-机组详情 |
| `market_wind_units` | 风电机组信息 | UnitWindGenerators | 5 | 企业信息(补充) |
| `market_solar_units` | 光伏机组信息 | UnitSolarGenerators | 4 | 企业信息(补充) |
| `market_loads` | 负荷数据 | Loads | 30 | 市场交易(参考) |
| `market_day_ahead_quotes` | 日前市场机组报价 | MarDayAheadUnitQuotes | 320 | 市场交易-机组报价 |
| `market_auxiliary_quotes` | 辅助服务市场报价 | MarAucillaryUnitQuotes | 32 | 市场交易-辅助服务申报 |

### 来源: ROTS_合并数据_按顺序.xlsx（成交历史汇总）

| 表名 | 说明 | 记录数 | 对应页面模块 |
|------|------|--------|-------------|
| `market_clearing_history` | 成交历史数据(5个指标×3936时段) | 5 | 成交历史 |

指标包括：出清电价、风电消纳功率、光伏消纳功率、总负荷功率、供需比

每条记录的 `period_values` 字段为JSON数组，存储所有时段的数值。

| 表名 | 说明 | 记录数 | 对应页面模块 |

| `market_out_results` | 出清结果时序数据(22个sheet) | 1215 | 信息披露、结算报告 |

| `wind_wt_quote_cost` | 风电报价成本 | 96 | 5 |
| `solar_pv_opera_power` | 光伏运行功率 | 4 | 96 |
| `solar_pv_curt_power` | 光伏弃光功率 | 4 | 96 |

## 三、数据导入

运行导入脚本：
```bash
cd backend
python -m app.services.import_market_data
```

导入脚本位置：`backend/app/services/import_market_data.py`
